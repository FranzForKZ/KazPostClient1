Комментарии разработчика бибилиотеки:
Выложенная версия собрана под .NET 3.5, x64. Завтра будет и версия х32. Для работы библиотеки также требуется установленный redistributable visual studio 2015.


Состав архива:
Tevian.FaceAnalyze.dll - C# - обертка, реализующая предложенный вами API
facesdk_c_static.dll - наша библиотека
data - папка с классификаторами, необходимыми для работы библиотеки
classifiers.ini - файл, в котором прописаны пути к классификаторам
config.txt - пример файла конфигурации
Все это для успешной работы должно находиться в одной папке.


В файле конфигурации строчки со значениями могут идти в любом порядке, названия параметров по идее не требуют дополнительных пояснений. 
Все размеры (регион интереса ROI, минимальное и максимальное расстояние между зрачками) задаются отдельно для анализа сканов и видео, в 
долях от размера кадра. На данный момент не целые числа принимаются только с десятичной запятой (вместо точки), но это завтра поправим.

Все параметры, передаваемые в функцию Init, пока игнорируются. Т.е. вся настройка сделана через файл конфигурации. Если все-таки нужно брать 
параметры, передаваемые в Init (но там все равно нет части необходимых нам параметров), то хотелось бы уточнить, что означают, например, 
yawThreshold и yawMean, зачем здесь второе значение?

При анализе скана возвращаются координаты всех найденных лиц и фильтры для лица с наиболее высокой уверенностью детектора.

При анализе кадра с камеры дополнительно возвращаем значение качества лица, оно записано в ImageDimensions: DimensionName="Качество", 
DimensionValue - значение качества в диапазоне от 0 до 1. Это оценка общего качества лица, она тем выше, чем лицо ближе к фронтальному, 
чем лучше освещено и меньше размыто. Таким образом, на вашей стороне можно выбрать кадр с лицом наилучшего качества за время сессии с камеры.

В поле Errors в текущей версии возвращаем ошибки, связанные с количеством лиц, слишком большие повороты и внутренние ошибки библиотеки. 
При отсутствии ошибок возвращается -1 (int, все биты единичные).
